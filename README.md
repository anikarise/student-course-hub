# Student Course Hub

This project is a web application developed for the **CTEC2712 Web Application Development** module.

The purpose of the system is to allow prospective students to explore available university programmes and register their interest in them. The university can then generate a mailing list to contact interested students.

---

## Features

- View available undergraduate and postgraduate programmes
- Register interest in a programme
- Store student contact details in a MySQL database
- Prevent duplicate registrations
- Admin page to view interested students
- Modern user interface with programme images and responsive design

---

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- GitHub

---

## Pages in the Application

| Page | Description |
|-----|-------------|
| index.php | Displays all available programmes |
| register.php | Allows students to register interest |
| process.php | Processes and stores student interest |
| admin.php | Allows administrators to view student interest list |
